Сайт, позволяющий проходить тесты , создавать их и удалять.
Также существует возможность создания тестов только для авторизованных пользователей и соотвественно их решения.
Если пользователь не авторизован, то он не сможет воспользоваться многими функциями сайта, такими как:

1. Удаление и добавление теста и его вопросов.
2. Решение тестов только для авторизованных пользователей.

Также присутствует возможность регистрации на сайте.

Добавление и удаление  тестов для авторизованных пользователей происходит только через админку, то есть, если у пользова
теля недостаточно прав, он этого сделать не сможет! Но сможет увидеть эти тесты и их решить!

На этом пока всё :)
Сайт в будущем будет дорабатываться!
